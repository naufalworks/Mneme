This is NOT an LLM - it's a storage/retrieval system.

How it works:
  1. You (or Claude) tell it what to store
  2. It stores using math (hypervectors) + files
  3. You (or Claude) query it later
  4. It returns stored information

No API calls. No external services. Just Python + NumPy.
